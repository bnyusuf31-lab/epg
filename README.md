# epg

`epg.xml` dosyalarıyla hangi kanalda hangi programın olduğunu ve yayın akışını kolayca öğrenebilirsiniz.

## 🚀 Özellikler

* **XML İşleme:** Standart XML formatındaki EPG verilerini hızlıca analiz eder.
* **Yayın Akışı:** Kanalların günlük veya anlık yayın programlarını listeler.
* **Detaylı Bilgi:** Program başlangıç-bitiş saatleri ve içerik açıklamalarını görüntüler.
